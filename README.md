 🔐 Self Sovereign Identity using Blockchain

A decentralized identity management system built using Ethereum Blockchain, Solidity, MetaMask, Ganache, and Remix IDE.

 🚀 Project Overview

This project implements a **Self Sovereign Identity (SSI)** system where users can:

- Register their identity securely
- Receive verifiable credentials
- Verify credentials on blockchain
- Maintain decentralized ownership of identity data

The project demonstrates how blockchain can eliminate centralized identity management systems and improve trust, privacy, and security.

 ⚙️ Workflow

1️⃣ Register Identity
Users register their identity through the smart contract.

 2️⃣ Issue Credential
Authorized issuer provides credentials to users.

 3️⃣ Verify Credential
Verifier validates the credential authenticity directly from blockchain.
